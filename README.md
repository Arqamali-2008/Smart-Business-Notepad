# Smart-Business-Notepad
A smart HTML, CSS, and JavaScript-based financial record management system for tracking cheque and loan transactions with local storage and print summaries.


AMI Business Notepad System is a lightweight financial record management web application designed for small businesses.

It allows users to manage:

Cheque records

Loan records

Date-based filtering (Day, Month, Year)

Print summaries

Persistent storage using browser LocalStorage

The system is fully built using:

HTML

CSS

JavaScript

No backend or database is required. All data is securely stored in the browser.

Developed for practical business use and financial tracking.
